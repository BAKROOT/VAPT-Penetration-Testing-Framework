# Methodology

This section defines the testing methodology used across all engagements. It aligns with recognized industry standards and provides a consistent, repeatable process from pre-engagement through reporting and retest.

## Core Phases

1. Pre-engagement & scoping
2. Reconnaissance (passive + active)
3. Scanning & enumeration
4. Vulnerability analysis
5. Exploitation (controlled)
6. Post-exploitation & impact analysis
7. Reporting
8. Remediation & retest

## Principles

- **Authorization first** — no testing without signed ROE.
- **Least impact** — favor non-destructive techniques; coordinate any high-risk testing.
- **Evidence-driven** — every finding is backed by reproducible evidence.
- **Repeatable** — same process, same quality, every engagement.
