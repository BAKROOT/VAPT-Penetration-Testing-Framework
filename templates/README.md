# Templates

Reusable deliverable templates used to produce consistent, professional engagement outputs.

## Contents

| File | Purpose |
| --- | --- |
| `pentest-report-template.md` | Full penetration test report structure |
| `executive-summary-template.md` | One-page leadership summary |
| `finding-writeup-template.md` | Standard format for a single finding |

## Usage

1. Copy the relevant template into your engagement workspace (outside this repo).
2. Fill in all `[bracketed]` placeholders.
3. Score each finding with CVSS v3.1 (see `findings/cvss-scoring-guide.md`).
4. Keep client-specific and sensitive content out of this repository.

> Templates are intentionally generic and sanitized. Never commit a completed, client-specific report here.
